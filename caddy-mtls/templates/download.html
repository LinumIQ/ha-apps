📱 How to Import Certificates

iOS / iPadOS

  1. Download the .p12 file (or scan QR code)
  2. Go to Settings → Profile Downloaded
  3. Install the profile and enter the certificate password
  4. Go to Settings → General → About → Certificate Trust Settings
  5. Enable full trust for the CA certificate

Android

  1. Download the .p12 file
  2. Go to Settings → Security → Encryption & credentials
  3. Tap "Install a certificate" → "VPN & app user certificate"
  4. Select the downloaded .p12 file and enter the password

Windows

  1. Download the .p12 file
  2. Double-click the file to open Certificate Import Wizard
  3. Select "Current User" and click Next
  4. Enter the certificate password
  5. Let Windows automatically select the certificate store

macOS

  1. Download the .p12 file
  2. Double-click to open in Keychain Access
  3. Enter the certificate password when prompted
  4. The certificate will be added to your login keychain

Linux (Firefox)

  1. Download the .p12 file
  2. Go to Settings → Privacy & Security → Certificates
  3. Click "View Certificates" → "Your Certificates" tab
  4. Click "Import" and select the .p12 file
  5. Enter the certificate password
